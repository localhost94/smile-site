PLATFORM INDEPENDENCE

Applicants must provide a description of how open-source components are independent and/or list the open alternatives for any closed component(s). Applications should be able to demonstrate that closed components can be replaced with open alternatives with minimal configuration changes and without requiring a major overhaul of the entire solution.

DOCUMENTATION

MECHANISM FOR EXTRACTING DATA

If Personally Identifiable information is stored then link(s) to the privacy policy, terms of service, or other relevant legal documentation must be provided
